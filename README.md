# تطبيق المصلي (Almosaly App)

تطبيق إسلامي متكامل لعرض مواقيت الصلاة والقرآن الكريم والأذكار والقبلة والأحاديث النبوية.

## الميزات الرئيسية

### 1. شاشة البداية ومعالج الإعداد الأولي
- شاشة ترحيبية بتصميم إسلامي جذاب
- معالج إعداد لتحديد الموقع:
  - تحديد تلقائي عبر GPS
  - تحديد يدوي باختيار الدولة والمدينة

### 2. الشاشة الرئيسية
- عرض مواقيت الصلاة لليوم الحالي
- وقت الصلاة القادمة مع عد تنازلي
- التاريخ الهجري
- تصميم إسلامي بألوان خضراء وزخارف إسلامية

### 3. القائمة الجانبية
تحتوي على الأقسام التالية:
- **الرئيسية**: عرض مواقيت الصلاة
- **القرآن الكريم**:
  - عرض سور القرآن كاملاً
  - إمكانية وضع علامات مرجعية
  - بحث داخل السور
  - تحميل السور للقراءة بدون إنترنت
- **الأذكار**:
  - مجموعة من الأذكار الصحيحة
  - عداد إلكتروني مع التقدم التلقائي
  - إمكانية التكرار المخصص
- **القبلة**:
  - بوصلة ديناميكية لتحديد اتجاه القبلة
  - تعمل بناءً على موقع المستخدم
- **الحديث**:
  - أحاديث نبوية صحيحة
  - خيار التحميل للاستخدام بدون إنترنت
- **الإعدادات**: تكوين التطبيق

### 4. الإعدادات
- **أصوات الأذان**: 10 خيارات لأشهر القراء
  - مشاري العفاسي
  - عبدالباسط عبدالصمد
  - محمد صديق المنشاوي
  - سعد الغامدي
  - ماهر المعيقلي
  - أحمد العجمي
  - خالد القحطاني
  - علي الحذيفي
  - صلاح البدير
  - عبدالرحمن السديس

- **الصامت التلقائي**:
  - تفعيل/تعطيل الصامت أثناء الصلاة
  - اختيار الصلوات المراد كتم الصوت فيها

- **إشعارات الذكر**:
  - تفعيل/تعطيل الإشعارات
  - تخصيص التكرار (كل 5-120 دقيقة)
  - 5 خيارات للأصوات

## التقنيات المستخدمة

- **Flutter**: إطار عمل متعدد المنصات
- **Provider**: لإدارة الحالة
- **adhan**: لحساب مواقيت الصلاة
- **geolocator**: لتحديد الموقع الجغرافي
- **hijri_calendar**: للتاريخ الهجري
- **audioplayers**: لتشغيل الأصوات
- **shared_preferences**: لحفظ الإعدادات

## التصميم الإسلامي

- استخدام خط "أميري" العربي
- ألوان خضراء مستوحاة من الفن الإسلامي
- زخارف ونقوش إسلامية
- أيقونات معبرة عن كل قسم

## التثبيت والتشغيل

1. تأكد من تثبيت Flutter SDK
2. استنساخ المشروع:
   ```bash
   git clone <repository-url>
   cd almosaly
   ```

3. تثبيت المكتبات:
   ```bash
   flutter pub get
   ```

4. تشغيل التطبيق:
   ```bash
   flutter run
   ```
